# Privacy Policy for Axelll Bot Center [wip]

Last Updated: 15 November 2025

1. This bot collects only the minimum data required for its features, including:
   - Server IDs
   - Channel IDs
   - User IDs (only when required)
   - Server configuration settings
   - Logs for moderation commands (if enabled)

2. This bot does NOT collect:
   - Message content (unless specifically required for a feature)
   - Personal user information
   - IP addresses
   - Payment or financial data

3. All collected data is used only for bot functionality and is never sold or shared with third parties.

4. Data is stored securely and only as long as necessary for the bot to operate.

5. Server owners can request deletion of their server data by contacting the bot developer.

6. This bot follows Discord’s Developer Terms of Service and all data handling rules.
